This is a repository for ECell authentication portal

Steps -
1. Clone the repository to your local environment
2. In /backend and /frontend, there exist files called e.env. Kindly rename the files as .env
3. npm install in /frontend and /backend
4. npm start in both directories as well
4. portal runs on 5000

The DB and the JWT secret are present which expire in 7 days
The use custom DB and JWT, please edit the .env files
